# Operation 01 - Intake and Evidence Gate

Do not begin decomposition during this operation.

1. Ask the user to select `propose` or `operate`.
2. Confirm the monolith path, approved corporate sources, objective, requested
   decomposition axis, target constraints, and required outputs.
3. Ask the questions in `templates/clarification-request.md`.
4. Create the evidence register from `templates/evidence-register.md`.
5. Record unavailable, contradictory, unsafe, or unauthorized sources as gaps.
6. Ask the user to resolve all blocking gaps.
7. Obtain explicit confirmation to begin current-state analysis.

The gate fails when authoritative sources, scope, vocabulary, decomposition axis,
acceptance criteria, or requested mode remain unclear.
